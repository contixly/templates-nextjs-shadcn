# Настройки аккаунта

Настройки аккаунта - это глобальная область управления пользователя. Она не привязана к одному
рабочему пространству и остается доступной, даже если у пользователя нет доступных рабочих
пространств.

В шаблоне есть отдельные разделы для профиля, подключенных OAuth-провайдеров, активных сессий,
личных приглашений в рабочие пространства, персональных API-ключей и удаления аккаунта.

## Разделы аккаунта

| Раздел | Маршрут | Для чего нужен |
| ------ | ------- | -------------- |
| Профиль | `/user/profile` | Проверить данные аккаунта и обновить отображаемое имя. |
| Подключения | `/user/connections` | Подключить или отключить настроенные OAuth-провайдеры. |
| Безопасность | `/user/security` | Посмотреть активные сессии и отозвать доступ. |
| Приглашения | `/user/invitations` | Посмотреть приглашения в рабочие пространства для текущего пользователя. |
| API-ключи | `/user/api-keys` | Управлять персональными API-ключами для `/api/v1`. |
| Опасные действия | `/user/danger` | Удалить текущий аккаунт после явного подтверждения. |

## Модель доступа

Все настройки аккаунта требуют входа. Серверные действия загружают текущую сессию на сервере,
проверяют ввод, выполняют изменение и обновляют затронутый кеш перед ответом UI.

## Когда использовать настройки аккаунта

Используйте настройки аккаунта для личных данных и персонального доступа. Используйте настройки
рабочего пространства, когда изменение относится к конкретному рабочему пространству на базе
организации: участникам, командам, API-ключам рабочего пространства или списку разрешенных доменов
email.
