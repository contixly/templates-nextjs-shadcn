# Удаление аккаунта

Страница опасных действий содержит необратимые операции с аккаунтом. В шаблоне основное такое
действие - удаление текущего пользователя.

## Перед удалением

Удаление аккаунта необратимо из UI. Перед продолжением проверьте подключенных провайдеров,
API-ключи, рабочие пространства и ожидающие приглашения.

Шаблон также отзывает персональные API-ключи удаляемого пользователя. Данные рабочих пространств
управляются правилами владения для рабочих пространств и организаций.

## Удалить аккаунт

1. Откройте `/user/danger`.
2. Прочитайте описание необратимого действия.
3. Введите email аккаунта точно так, как требуется.
4. Подтвердите удаление.

Действие отклоняет несовпадающий ввод подтверждения. Это защищает от случайного удаления обычным
кликом по кнопке.

## Заметка для расширения продукта

Если ваш сервис хранит пользовательские продуктовые данные, расширьте этот сценарий правилами
хранения, экспорта и передачи данных. Покажите эти правила на странице опасных действий до
подтверждения удаления.
