# Документация шаблона

Этот раздел показывает, как в шаблоне устроена публичная документация на Markdown и MDX. Страницы
собираются из `src/features/documents-system/content`, попадают в левую навигацию, индексируются для
поиска и рендерятся внутри обычного App Router маршрута `/docs`.

## Как расширять документацию

Добавляйте новые страницы в `src/features/documents-system/content`. Каждая страница должна иметь
frontmatter с `title`, `description`, `group`, `parentItem`, `order`, `status` и `toc`.

<Callout title="Публикация" variant="info">
  В production отображаются только страницы со статусом `published` или `archived`, если у них не
  стоит `hide: true`. В локальной разработке видны черновики, review-страницы и authoring-примеры.
</Callout>
