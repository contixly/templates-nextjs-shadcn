# Еженедельное обновление - 6-13 апреля 2026

## ✨ Новые возможности

- **Первый публичный шаблон приложения**: опубликована базовая версия Next.js-шаблона с App Router, React 19, TypeScript, Tailwind CSS v4, shadcn/ui, Prisma и Better Auth.

- **Авторизация и страницы аккаунта**: добавлены публичная главная, защищённая оболочка приложения, маршруты входа, профиль, подключённые аккаунты, управление сессиями, удаление аккаунта и health check endpoint.

- **Каркас рабочих пространств**: добавлены стартовые сценарии создания, загрузки, обновления и удаления workspace, а также dashboard и навигация для авторизованной части приложения.

## 🔧 Улучшения

- **Главная страница под шаблон**: тексты и структура главной страницы описывают рабочие пространства, авторизацию и переиспользуемый каркас приложения вместо placeholder-контента.

- **Более устойчивое переключение темы**: улучшено поведение theme switcher при hydration, чтобы интерфейс предсказуемо отображался до появления состояния темы в браузере.

- **Более чистый поток данных workspace**: убраны placeholder-счётчики и упрощена логика репозиториев и компонентов на стартовых экранах рабочих пространств.

## 🐛 Исправления

- **Стабильность первого build**: исправлены ранние проблемы маршрутов и импортов, найденные в проверках сборки, включая именование dashboard route и fallback для Prisma tooling.

## 📝 Документация

- Добавлены стартовые README, checklist для использования шаблона, пример окружения, contribution guide, security policy и release notes для первой публичной версии шаблона.
