# Еженедельное обновление - 6-13 апреля 2026

## ✨ Новые возможности

- **Первый публичный шаблон приложения**: опубликована базовая версия Next.js-шаблона с App Router, React 19, TypeScript, Tailwind CSS v4, shadcn/ui, Prisma и Better Auth.

- **Авторизация и страницы аккаунта**: добавлены публичная главная, защищенная оболочка приложения, маршруты входа, профиль, подключенные аккаунты, управление сессиями, удаление аккаунта и маршрут проверки работоспособности.

- **Каркас рабочих пространств**: добавлены стартовые сценарии создания, загрузки, обновления и удаления рабочих пространств, а также панель и навигация для авторизованной части приложения.

## 🔧 Улучшения

- **Главная страница под шаблон**: тексты и структура главной страницы описывают рабочие пространства, авторизацию и переиспользуемый каркас приложения вместо временного контента.

- **Более устойчивое переключение темы**: улучшено поведение переключателя темы при гидратации, чтобы интерфейс предсказуемо отображался до появления состояния темы в браузере.

- **Более чистый поток данных рабочего пространства**: убраны временные счетчики и упрощена логика репозиториев и компонентов на стартовых экранах рабочих пространств.

## 🐛 Исправления

- **Стабильность первой сборки**: исправлены ранние проблемы маршрутов и импортов, найденные в проверках сборки, включая именование маршрута панели и запасной сценарий для локальных инструментов Prisma.

## 📝 Документация

- Добавлены стартовый README, контрольный список для использования шаблона, пример окружения, руководство по участию, политика безопасности и заметки к релизу для первой публичной версии шаблона.
