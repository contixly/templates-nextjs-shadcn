# Еженедельное обновление - 13-20 апреля 2026

## ✨ Новые возможности

- **Локализация приложения**: добавлена основа локализации на `next-intl`, каталоги сообщений на английском и русском, конфигурация локалей и typed namespaces для страниц.

- **Переведённые основные сценарии**: локализованы главная страница, страницы авторизации, настройки аккаунта, рабочие пространства, dashboard, навигация, breadcrumbs, диалоги, формы и общие состояния ошибок.

- **Локализованные metadata страниц**: title, description, Open Graph и Twitter metadata теперь берутся из тех же namespace, что и видимый интерфейс.

## 🔧 Улучшения

- **Чище runtime-конфигурация**: обновлена клиентская настройка аналитики и обработка global error, чтобы локализованные страницы и fallback-экраны использовали безопасные runtime-значения.

- **Единый источник данных для страниц**: route definitions теперь хранят page key и namespace переводов, поэтому headers, breadcrumbs и metadata читают одинаковую структуру.

## 🐛 Исправления

- **Регрессии локализованного UI**: добавлены проверки переведённых форм, metadata exports, runtime components и page translation consumers, чтобы в основных сценариях не возвращались hard-coded тексты.

## 📝 Документация

- Добавлены release notes для первого шаблона и для релиза локализации. Зафиксированы design и implementation plan для будущих изменений, завязанных на локали.
