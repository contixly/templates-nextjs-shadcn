# Еженедельные изменения

Этот раздел собирает пользовательские изменения шаблона по неделям. Заметки основаны на истории
репозитория, заметках к релизам, требованиях OpenSpec и публичной документации.

## Выпуски по неделям

- [Еженедельное обновление - 29 июня-6 июля 2026](/docs/history/change-logs/2026-07-06-weekly-changelog) - API-ключи, API v1, публичная документация, локализованный контент документации и расширенное E2E-покрытие.
- [Еженедельное обновление - 27 апреля-4 мая 2026](/docs/history/change-logs/2026-05-04-weekly-changelog) - команды рабочих пространств, приглашения в команды, распределенное кэширование, настроенные OAuth-провайдеры, локальная авторизация для автоматизации и Playwright E2E.
- [Еженедельное обновление - 20-27 апреля 2026](/docs/history/change-logs/2026-04-27-weekly-changelog) - рабочие пространства на базе организаций, настройки, приглашения, усиление безопасности и политики разрешенных email-доменов.
- [Еженедельное обновление - 13-20 апреля 2026](/docs/history/change-logs/2026-04-20-weekly-changelog) - локализация приложения, локализованные метаданные и документация релизных заметок.
- [Еженедельное обновление - 6-13 апреля 2026](/docs/history/change-logs/2026-04-13-weekly-changelog) - первая публичная основа шаблона с авторизацией, рабочими пространствами, документацией по настройке и стартовым UI.
