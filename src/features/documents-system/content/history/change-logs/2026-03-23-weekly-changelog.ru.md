# Еженедельное обновление — 16–23 марта 2026

## ✨ Новые возможности

- **Приватный режим**: Добавлена функциональность приватного режима на страницы событий и дашборда с фильтрацией рабочих пространств и контролем доступа.

- **Интеграция с сервером отчётов**: Реализована полноценная работа с сервером отчётов:
  - Новые API-эндпоинты для создания и управления задачами отчётов
  - Получение одной задачи и списка задач с фильтрацией и кэшированием
  - API метаданных сервера отчётов с выделенным контроллером
  - Полные типы для входных и выходных данных задач

- **Улучшенные цвета тем**: Внедрён новый генератор цветов тем на основе пастельных OKLCH-оттенков с детерминированной генерацией и проверкой контрастности для доступности.

## 🔧 Улучшения

- **Улучшения таблиц данных**:
  - Фиксированные заголовки с синхронизированной горизонтальной прокруткой
  - Улучшенные размеры колонок для таблиц событий, проектов, площадок, сезонов и отчётов
  - Исправлен расчёт rowspan для корректного отображения заголовков
  - Улучшен макет заголовков колонок с доступностью и отступами

- **Поведение прокрутки**: Улучшена прокрутка к началу страницы при смене маршрута с учётом контейнера.

- **Валидация форм**: Усилена валидация строк с улучшенной обработкой ошибок и проверкой HTML-контента.

- **Управление расписанием**:
  - Унифицирован рендеринг участников в компонентах расписания
  - Добавлены предупреждения о статусах черновика и редактирования для контента
  - Оптимизировано извлечение тем с использованием useMemo

- **Система типов**: Переработан ServerResponse как обобщённый mapped type с сохранением обратной совместимости.

- **Качество кода**:
  - Централизована логика предупреждений предложений и утилиты ссылок
  - Заменён unstable_cache на React cache
  - Сделаны поля контента опциональными с улучшенной обработкой инвариантов

## 🐛 Исправления

- **Коды ответов API**: Исправлены HTTP-статусы для отсутствующих ID активностей и событий с 404 на 400 Bad Request.

- **Персистентность Redux**: Удалён 'api' из чёрного списка персистентности для корректного сохранения в IndexedDB.

- **Утилиты кэширования**: Исправлена обработка типов тегов кэша для поддержки опциональных параметров в мутациях.

- **Контроль доступа к событиям**: Добавлена проверка приватного режима для пункта меню пользователей события.

- **Русская локализация**: Исправлен предлог в русском переводе описания сервера отчётов.

- **Макет документа**: Исправлен класс максимальной ширины для адаптивного заголовка документа.
