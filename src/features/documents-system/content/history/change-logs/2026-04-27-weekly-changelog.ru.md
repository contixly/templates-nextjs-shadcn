# Еженедельное обновление - 20-27 апреля 2026

## ✨ Новые возможности

- **Рабочие пространства на организациях**: рабочие пространства теперь работают поверх организаций Better Auth. Добавлены маршруты `/w/:organizationKey`, ограниченные конкретной организацией.

- **Настройки и совместная работа в рабочем пространстве**: появились страницы настроек рабочего пространства, пользователей, приглашений, команд и ролей, а также каталог участников, управление ролями, создание приглашений, просмотр приглашений и онбординг для пользователя без рабочего пространства.

- **Политики разрешенных email-доменов**: администратор рабочего пространства может задать разрешенные домены, применять их к приглашениям и видеть предупреждения, если текущие участники находятся вне активной политики.

## 🔧 Улучшения

- **Безопасные настройки по умолчанию**: добавлены заголовки безопасности браузера, HSTS для боевой среды, строгая политика внешних изображений, безопасная обработка перенаправлений авторизации и нормализованная настройка публичного URL приложения.

- **Приватность приглашений и участников**: детали приглашения скрываются, если текущий пользователь не является адресатом, повторные ожидающие приглашения блокируются, а принятие приглашения переключает активное рабочее пространство.

- **Защита идентичности рабочего пространства**: slug проверяется среди всех организаций, а переименование рабочего пространства больше не меняет slug без явной отправки нового значения.

## 🐛 Исправления

- **Граница защищенных действий**: защищенные серверные действия теперь используют проверенную сессию Better Auth и не доверяют переданным заголовкам с идентификатором пользователя.

- **Безопасность списка сессий**: список сессий аккаунта больше не отдает в клиентский UI сырые токены сессий Better Auth.

## 📝 Документация

- Добавлены заметки к релизам по рабочим пространствам на базе организаций, усилению безопасности и ограничениям доменов. Обновлены README, заметки по настройке и требования OpenSpec для управления рабочими пространствами и приглашениями.
