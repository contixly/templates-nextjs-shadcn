# Еженедельное обновление - 29 июня-6 июля 2026

## ✨ Новые возможности

- **Личные API-ключи и API-ключи рабочего пространства**: добавлены страницы управления API-ключами в `/user/api-keys` и `/w/:organizationKey/settings/api-keys` с созданием, редактированием, отключением, удалением, одноразовым показом секрета, сроком действия, ограничением частоты и наборами прав.

- **Доступ к API v1 по областям доступа**: добавлены стартовые маршруты `/api/v1` для метаданных ключа, организаций, деталей организации, участников, команд и участников команды. API-клиенты используют заголовок `x-api-key`, а не браузерную сессию.

- **Публичная система документации**: добавлен `/docs` с общей оболочкой, боковой навигацией, хлебными крошками, поиском, рендерингом MDX, оглавлением, метаданными страниц, ссылками на заголовки, кнопками копирования кода и изображениями Open Graph.

- **Локализованный контент документации**: файлы документации теперь могут использовать суффиксы локали, например `.en.md` и `.ru.md`; реестр выбирает вариант под текущий язык UI и помечает запасной контент, если страница есть только на одной локали.

## 🔧 Улучшения

- **E2E-покрытие и надежность автоматизации**: расширено Playwright-покрытие на базе OpenSpec для аккаунта, рабочего пространства, приглашений, команд, API-ключей, локальной автоматизации, безопасности и `/api/v1`. Общие вспомогательные функции делают браузерные запуски повторяемее.

- **Единая конфигурация API-ключей**: идентификаторы конфигурации API-ключей и константа заголовка `x-api-key` централизованы, при этом существующие импорты остаются совместимыми.

- **Удобство подготовки документации**: добавлены примеры авторинга, карточки ссылок, выделенные блоки, вкладки, блоки дерева файлов, локализованные подписи MDX UI, проверка битых ссылок, проверки канонических URL и поисковая индексация страниц документации.

- **Полировка навигации документации**: добавлена ссылка на документацию в навигацию приложения, удалена старая общая ссылка помощи, локализованы подписи UI документации и выровнена типографика с остальным приложением.

## 🐛 Исправления

- **Регрессии локалей документации**: исправлены запасной вариант локали по умолчанию, генерация канонического slug, обнаружение дублирующихся локалей, проверка ссылок по локализованным файлам, отсутствующие сообщения `documentsSystem.pages.home` и локализованные подписи времени чтения.

- **Согласованность сессий автоматизации**: добавлена возможность отключать кеш cookie сессии для локальной автоматизации и запусков Playwright против уже поднятого сервера, чтобы тесты читали свежее состояние сессии.

- **Обновление маршрутов рабочего пространства**: создание рабочего пространства и изменения настроек теперь обновляют ключевые пути и переводят изменения slug или ключа маршрута на канонический маршрут настроек.

## 📝 Документация

- Добавлены заметки к релизам по управлению API-ключами и E2E-покрытию, спецификации OpenSpec для API-ключей, API v1, локальной автоматизации, сценариев аккаунта и сессий, запасных сценариев рабочего пространства и системы документации, а также публичные страницы для авторов документации в `/docs`.
