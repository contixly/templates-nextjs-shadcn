# История версий

Этот раздел описывает опубликованные версии шаблона. В заметки к версиям попадают изменения,
которые влияют на пользователей шаблона: новые экраны, сценарии авторизации, потоки рабочих пространств,
API-доступ, локализация, безопасность и проверки качества.

## Актуальная версия

- [v0.0.11](/docs/history/releases/0.0.11) — публичный раздел документации, страницы Markdown/MDX
  на русском и английском языках, навигация `/docs`, поиск по документации, страницы истории версий
  и еженедельных изменений.

## Архив версий

| Версия | Основная тема |
| ------ | ------------- |
| [v0.0.11](/docs/history/releases/0.0.11) | Публичный раздел документации, языковые варианты страниц и поиск по документации. |
| [v0.0.10](/docs/history/releases/0.0.10) | E2E-покрытие и надежность сценариев автоматизации. |

| [v0.0.8](/docs/history/releases/0.0.8) | Локальная авторизация для автоматизации, Playwright E2E и хуки для политики приглашений. |

| [v0.0.4](/docs/history/releases/0.0.4) | Усиление безопасности, безопасные перенаправления, приватность приглашений и защита данных сессий. |
| [v0.0.3](/docs/history/releases/0.0.3) | Рабочее пространство на базе организаций Better Auth, приглашения, роли и страницы настроек. |

| [v0.0.1](/docs/history/releases/0.0.1) | Первая публичная основа шаблона Next.js приложения. |

## Как читать историю

Используйте историю версий, чтобы понять крупные опубликованные возможности. Более частые изменения
смотрите в [еженедельных заметках](/docs/history/change-logs). Если нужна детальная техническая
история, в репозитории также сохранены текстовые заметки к версиям в `docs/releases/template`.
