# v0.0.3

v0.0.3 заменяет раннюю workspace-only основу на Better Auth Organizations, сохраняя "Workspace" как
product-facing term. Релиз добавляет organization-scoped routing, workspace settings, invitations,
member management, role-aware permissions и zero-workspace onboarding.

## Главное

- Добавлена Better Auth organization support для workspace collaboration.
- Добавлены organization-scoped routes `/w/:organizationKey/...`.
- `/dashboard` получил active и fallback organization resolution.
- Добавлены workspace settings pages для details, users, invitations, teams и roles.
- Добавлены invitation creation, listing, link copying, accept, reject и pending-invitation entry
  points.
- Добавлены member directory, direct member add и owner/admin/member role management.
- Добавлен onboarding для пользователей без доступных workspaces.

## Что изменилось для пользователей шаблона

Workspace URLs теперь предпочитают `/w/:organizationKey/...`, где key может быть organization slug
или id. Workspace switchers меняют active organization context только через explicit user actions и
по возможности сохраняют matching base workspace routes.

Пользователи могут создавать и обновлять workspaces, управлять members и roles при наличии прав,
создавать invitations и просматривать personal pending invitations в account или welcome surfaces.

## Upgrade notes

- Workspaces теперь backed by Better Auth organizations вместо standalone workspace model.
- Existing deployments должны выполнить новые Prisma migrations перед использованием organization и
  invitation flows.
- Invitation acceptance требует, чтобы verified primary email invitee совпадал с invitation.
- Workspace permissions следуют Better Auth organization permission checks и built-in role rules.

## Качество

Релиз добавляет тесты для organization route guards, slug generation, repository cache tags,
workspace creation, updates, deletion, member management, role assignment, invitations, settings
pages, onboarding guards, sidebar switchers, account invitations и localized metadata consumers.
