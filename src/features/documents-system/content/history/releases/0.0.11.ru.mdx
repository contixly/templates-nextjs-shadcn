# v0.0.11

v0.0.11 превращает `/docs` в полноценный публичный раздел документации шаблона. В версии появились
страницы Markdown и MDX на русском и английском языках, поиск по документации, страницы истории
версий и еженедельных изменений, а также требования OpenSpec для механизма документации.

<Callout title="Фокус версии" variant="info">
  Эта версия помогает быстрее находить нужную информацию. Пользователи шаблона могут читать
  документацию по продуктовым сценариям, API, рабочим пространствам, разработке, версиям и
  еженедельным изменениям в одном публичном разделе.
</Callout>

## Главное

- Добавлен публичный раздел `/docs` с общей оболочкой, боковым меню, хлебными крошками, действиями
  страницы, оглавлением, переключением темы, каноническими маршрутами и изображениями для ссылок.
- Добавлен выбор страниц Markdown и MDX по языку интерфейса: русские и английские варианты сохраняют
  стабильные канонические URL, а при отсутствии нужного языка используется предсказуемый запасной
  вариант.
- Опубликована двуязычная документация для управления аккаунтом, API-доступа, настроек приложения,
  рабочих пространств, материалов для разработчиков, истории версий и еженедельных изменений.
- Добавлен поиск `/api/v1/documents-system/search` по страницам и заголовкам с учетом выбранного
  языка.
- Добавлены компоненты MDX для выделенных блоков, вкладок, сеток ссылок, копирования кода, якорей
  заголовков, локальных изображений и безопасных внутренних ссылок.

## Заметки по обновлению

- Миграция Prisma для этой версии не требуется.
- Метаданные пакета показывают `0.0.11`.
- В шаблон добавлены зависимости для MDX и `sharp` для локальных страниц документации и изображений.
- `next.config.ts` теперь включает расширения `.md` и `.mdx` для страниц и подключает `@next/mdx`.
- Публичная документация хранится в `src/features/documents-system/content/`.
- Языковые варианты страниц используют суффиксы `.en` и `.ru`, а внутренние ссылки остаются
  каноническими и начинаются с `/docs/...`.
- `/api/v1/documents-system/search` является публичным маршрутом поиска и не требует браузерной
  сессии.

## Качество

В версии добавлены проверки Jest для загрузки реестра страниц, разбора языка, выбора запасного
варианта, проверки ссылок, поиска, метаданных страницы, данных бокового меню, оглавления и
отслеживания прокрутки. Также добавлены требования OpenSpec для публичных маршрутов, оболочки,
метаданных реестра, языковых вариантов, правил видимости, навигации, статических параметров,
MDX-отрисовки, внутренних ссылок, поиска, изображений и режимов кеширования.
