# v0.0.1

v0.0.1 - первый публичный релиз Next.js application template. Он задает reusable baseline для
сервисов на Next.js, Prisma, Better Auth, Tailwind CSS v4 и shadcn/ui.

## Главное

- Next.js 16, React 19 и TypeScript starter с App Router structure.
- Tailwind CSS v4 и shadcn/ui component foundation.
- Prisma и PostgreSQL setup с generated client и initial migration.
- Better Auth integration с social login scaffolding.
- Feature Slice Design structure для business logic.
- Shared layout, navigation, metadata, sitemap, robots, Open Graph и Twitter image setup.
- Example authenticated application shell с dashboard, workspace и account areas.

## Что изменилось для пользователей шаблона

Первый релиз дает working application foundation, а не пустой starter. В него входят public landing
и welcome flows, protected route handling, health endpoint, reusable UI primitives, Better Auth
sessions, account settings и workspace repository/action patterns.

Также добавлена документация для contributors, security, environment и post-template setup, чтобы
новое приложение можно было настроить после генерации.

## Setup notes

Перед использованием шаблона в реальном проекте настройте:

- `DATABASE_URL`
- `BETTER_AUTH_SECRET`
- `BETTER_AUTH_URL`
- `PUBLIC_BASE_URL`
- OAuth provider credentials для провайдеров, которые планируете включить

Затем следуйте `TEMPLATE.md` для recommended setup sequence.

## Качество

Релиз включает Jest coverage для core UI и configuration regressions, а также follow-up fixes для
dashboard route imports, Prisma local tooling fallback, workspace data flow, theme hydration и
template messaging.
