# v0.0.1

v0.0.1 - первый публичный релиз шаблона приложения на Next.js. Он задает переиспользуемую основу
для сервисов на Next.js, Prisma, Better Auth, Tailwind CSS v4 и shadcn/ui.

## Главное

- Стартовый проект на Next.js 16, React 19 и TypeScript со структурой App Router.
- Tailwind CSS v4 и основа компонентов shadcn/ui.
- Настройка Prisma и PostgreSQL со сгенерированным клиентом и начальной миграцией.
- Интеграция Better Auth с каркасом OAuth-входов.
- Структура функциональных срезов для бизнес-логики.
- Общий макет, навигация, метаданные, sitemap, robots и настройка изображений Open Graph и Twitter.
- Пример защищенной оболочки приложения с панелью, рабочим пространством и разделами аккаунта.

## Что изменилось для пользователей шаблона

Первый релиз дает рабочую основу приложения, а не пустую заготовку. В него входят публичная главная,
экран приветствия, обработка защищенных маршрутов, маршрут проверки работоспособности,
переиспользуемые UI-примитивы, сессии Better Auth, настройки аккаунта и паттерны репозиториев и
действий рабочего пространства.

Также добавлена документация для участников разработки, безопасности, окружения и настройки после
создания проекта из шаблона.

## Заметки по настройке

Перед использованием шаблона в реальном проекте настройте:

- `DATABASE_URL`
- `BETTER_AUTH_SECRET`
- `BETTER_AUTH_URL`
- `PUBLIC_BASE_URL`
- учетные данные OAuth-провайдеров, которые планируете включить

Затем следуйте `TEMPLATE.md` для рекомендуемой последовательности настройки.

## Качество

Релиз включает покрытие Jest для основного UI и регрессий конфигурации, а также последующие
исправления импортов маршрута панели, запасного сценария локальных инструментов Prisma, потока
данных рабочего пространства, гидратации темы и текстов шаблона.
