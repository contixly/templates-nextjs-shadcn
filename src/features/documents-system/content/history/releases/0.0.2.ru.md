# v0.0.2

v0.0.2 добавляет reusable localization foundation для основных flows шаблона. В релиз входят
`next-intl`, English и Russian message catalogs, localized metadata и переведенный core UI copy.

## Главное

- `next-intl` добавлен в template stack.
- Добавлены English и Russian message catalogs для основных feature areas.
- Локализованы page titles, descriptions, Open Graph metadata и Twitter metadata.
- Core UI copy перенесен в translation namespaces вместо hard-coded strings.
- Добавлены shared helpers для server и client translation consumers.
- Environment examples обновлены для locale и analytics configuration.

## Что изменилось для пользователей шаблона

В шаблоне появился default bilingual setup с поддержкой `en` и `ru`. Route metadata, forms,
navigation, auth pages, account settings, workspace management screens и shared UI copy читают
тексты из message catalogs.

Default locale управляется через `PUBLIC_DEFAULT_LOCALE`.

## Upgrade notes

- `next-intl` входит в default stack.
- Translation files находятся в `src/messages/`.
- Default locale задается через `PUBLIC_DEFAULT_LOCALE`.
- Yandex Metrika теперь ожидает `NEXT_PUBLIC_YM_COUNTER_ID`.

## Качество

Релиз добавляет тесты для i18n request setup, page translation helpers, localized metadata,
translated runtime components, localized forms, schema-backed validation copy, environment access и
global error behavior.
