# v0.0.7

v0.0.7 добавляет опциональное кэширование на Redis/Valkey для Cache Components и ISR, улучшает
состояния загрузки рабочих пространств и скрывает OAuth-провайдеры входа, пока они не настроены
полностью.

## Главное

- Добавлены опциональные обработчики кеша Redis/Valkey для Next.js Cache Components и ISR.
- Добавлены общие настройки удаленного кеша, обработка URL подключения и префиксы ключей.
- Добавлен локальный запасной кеш в памяти, когда удаленное кэширование отключено.
- Добавлены границы Suspense для рабочего пространства и панели с заглушками загрузки под форму маршрутов.
- Добавлены примитивы боковой навигации для доступа к панели, создания рабочего пространства, дополнительных ссылок и меню аккаунта пользователя.
- Better Auth получает и показывает только полностью настроенные OAuth-провайдеры.

## Что изменилось для пользователей шаблона

Локальная разработка в одном экземпляре использует локальный запасной кеш. Развертывания в
нескольких экземплярах могут включить `REMOTE_CACHING_ENABLED=true` с `REDIS_URL` или `VALKEY_URL`
и уникальным `REMOTE_CACHING_PREFIX`.

Страницы рабочего пространства получили более стабильные состояния загрузки для панели,
участников, команд, приглашений, ролей и навигации настроек. Экраны входа и подключения аккаунтов
больше не показывают OAuth-провайдеров с неполным набором переменных окружения.

## Заметки по обновлению

- Настройте `REDIS_URL` или `VALKEY_URL` до включения удаленного кэширования.
- Используйте уникальный `REMOTE_CACHING_PREFIX` для каждого приложения или окружения при общем сервисе Redis/Valkey.
- Учетные данные OAuth-провайдеров должны быть полными; частично настроенные провайдеры остаются скрытыми.
- Встроенный кеш Next.js в памяти отключен в пользу настроенных обработчиков кеша.

## Качество

Релиз добавляет тесты для конфигурации кеша, разбора настроек Redis/Valkey, фильтрации
провайдеров, видимости провайдеров входа, доступности отвязки подключения, поведения навигации,
переключателей рабочих пространств, потоковых границ, поведения оболочки настроек и рендеринга
заголовка документа.
