# Рабочее пространство

Эта страница служит стартовым каркасом для документации рабочих пространств. В шаблоне уже есть
модель организаций, команд и настроек, поэтому продуктовая документация может расширять этот раздел
без изменения маршрутов docs-system.

## Что описывает страница

Здесь можно описывать базовые сценарии рабочего пространства: создание организации, переключение
контекста, приглашение участников, настройку команд и выдачу API-ключей.

## Когда использовать

Используйте эту страницу как родительский раздел, когда новая продуктовая функция относится к
организации, команде или настройкам workspace.
