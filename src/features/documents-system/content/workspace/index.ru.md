# Рабочее пространство

Рабочее пространство - это пользовательское название рабочей области приложения. В шаблоне каждое
рабочее пространство использует организацию Better Auth, которая хранит участников, роли, команды,
приглашения, API-ключи и настройки доступа.

Эта страница - общая карта раздела. Для конкретных задач используйте связанные страницы.

## Модель

| Понятие | Значение |
| ------- | -------- |
| Рабочее пространство | Пользовательская область совместной работы в продукте. |
| Организация | Модель Better Auth, которая хранит участие и данные доступа рабочего пространства. |
| Ключ организации | Slug или id, используемый в маршрутах `/w/:organizationKey/...`. |
| Активное рабочее пространство | Рабочее пространство, выбранное в текущей сессии пользователя. |
| Команда | Явная подгруппа внутри организации рабочего пространства. |

Маршруты под `/w/:organizationKey/...` проверяют доступ до рендера. Глубокие ссылки используют
рабочее пространство из URL и не переписывают активное рабочее пространство пользователя без явного
переключения контекста.

## Что могут делать пользователи

- Создавать рабочие пространства и переключать их контекст.
- Управлять деталями рабочего пространства и списком разрешенных доменов email.
- Смотреть участников и менять назначаемые роли.
- Создавать приглашения и копировать ссылки приглашений.
- Принимать или отклонять приглашения после входа.
- Создавать явные команды и управлять участием в командах.
- Создавать API-ключи рабочего пространства, если это разрешено.
